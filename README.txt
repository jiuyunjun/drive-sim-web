使用方法：
1. 用本地静态服务器打开 index.html。
   例如：
   - VS Code Live Server
   - Python: python -m http.server 8000
2. 浏览器访问 http://localhost:8000/drive-sim/ 或对应目录。
3. 点击“上传地图”，选择你的驾照考试中心平面图。
4. 通过“地图宽度（米）”或“拖拽地图比例”大致设置实际尺寸，系统会按图片比例自动计算高度，地图最大宽度支持到 1000 米。
5. 用 W A S D 或方向键驾驶，Q / E 控制左右转向灯。
6. 把车停到你想要的位置后，点击“设为起点”，之后重置车辆会回到这个位置和朝向。
7. 地图、地图比例、最大车速、起始位置等设置会自动保存在浏览器本地。

当前版本已实现：

- 基础加速、转向、倒车、手刹
